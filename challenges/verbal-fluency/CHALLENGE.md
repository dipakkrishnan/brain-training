# Verbal Fluency Challenge

## Goal

Create a short brain-training game that exercises verbal fluency.

## Cognitive Skill

Primary skill: verbal fluency.

Secondary skills may include vocabulary recall, creative association, and rapid constraint satisfaction.

## Player Experience

The game should be playable in 3-5 minutes. It may be solo or competitive, but the core loop should be understandable quickly.

## Requirements

- Include clear rules, scoring, and a complete play loop.
- Reward creative or skillful word production, not only typing speed.
- Include constraints that create interesting tradeoffs.
- The game must be playable without account creation or paid services.
- The submission should provide either a playable URL or one-command local run instructions.

## Evaluation Questions

Human evaluators will compare games from this challenge using questions like:

- Which game made wordplay feel more interesting?
- Which game better exercises verbal fluency?
- Which game had the better scoring incentives?
- Which game would you rather replay?
- Which game felt more novel?
