# Working Memory Challenge

## Goal

Create a short brain-training game that exercises working memory.

## Cognitive Skill

Primary skill: working memory.

Secondary skills may include attention control, sequence tracking, and pattern recall.

## Player Experience

The game should be playable by one person in 3-5 minutes. A new player should understand the basic loop within 60 seconds.

## Requirements

- Include clear rules, scoring, and a complete play loop.
- Include at least one difficulty progression or adaptive difficulty mechanism.
- Skill should matter more than luck.
- The game must be playable without account creation or paid services.
- The submission should provide either a playable URL or one-command local run instructions.

## Evaluation Questions

Human evaluators will compare games from this challenge using questions like:

- Which game would you rather play again?
- Which game better exercises working memory?
- Which game felt more intellectually interesting?
- Which game had clearer rules?
- Which game felt more novel?
