# Probabilistic Reasoning Challenge

## Goal

Create a short brain-training game that exercises probabilistic reasoning.

## Cognitive Skill

Primary skill: probabilistic reasoning.

Secondary skills may include risk assessment, expected-value thinking, and decision-making under uncertainty.

## Player Experience

The game should be playable in 3-7 minutes. It may be solo or competitive, but the evaluation artifact must be easy for a single evaluator to try.

## Requirements

- Include clear rules, scoring, and a complete play loop.
- Include uncertainty, hidden information, or stochastic outcomes.
- Good play should require better judgment than random guessing.
- The game must be playable without account creation or paid services.
- The submission should provide either a playable URL or one-command local run instructions.

## Evaluation Questions

Human evaluators will compare games from this challenge using questions like:

- Which game created more satisfying decisions?
- Which game better exercises probabilistic reasoning?
- Which game balanced luck and skill more effectively?
- Which game would you rather replay?
- Which game felt more novel?
