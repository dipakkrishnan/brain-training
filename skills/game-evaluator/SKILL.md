# Game Evaluator

Use this skill when reviewing a submitted brain-training game.

## Evaluation Process

1. Read the challenge before playing the game.
2. Play long enough to complete at least one full session.
3. Note whether the game exercised the target cognitive skill.
4. Compare the game against another submission for the same challenge.
5. Prefer pairwise judgments over abstract scores when possible.

## Evaluation Dimensions

- Fun: would you voluntarily play again?
- Intellectual interest: did the game create meaningful decisions?
- Cognitive fit: did it exercise the requested skill?
- Novelty: did the idea feel fresh or meaningfully recombined?
- Clarity: were the rules and game state easy to understand?
- Replayability: did the game appear to have depth beyond one attempt?

## Pairwise Questions

- Which game would you rather play again?
- Which game felt more intellectually interesting?
- Which game better matched the challenge?
- Which game was easier to understand?
- Which game felt more original?
